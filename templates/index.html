ShowMap


This section will contain a full map of Australia with leaflet js marker clusters. When users filter a location by state, the map will zoom in to that area. Users can also zoom in on the map which will scatter the markers based on the events lats and longs.

Event markers will have different colours according to event category. Each event will also have a popup containing event name and venue name.




ShowChart


This section will contain a bubble chart that shows the filtered events. Each event bubble is represented by that particular event's rank.




ShowTable


This section will contain a table of the filtered data. The columns will be Event Name, Event Description, Event Address.
